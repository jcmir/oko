# Инструкция по тестированию Трей-агента и Ядра на Windows

Так как система построена на защищенных каналах Windows (Named Pipes) и требует прав для управления процессами, тестирование нужно проводить по определенному алгоритму.

## Предварительная подготовка (Разово)
1. Убедитесь, что вы скомпилировали проект: `cargo build`
2. Откройте консоль PowerShell от имени Администратора и задайте мастер-пароль:
   `.\target\debug\core-service.exe --set-password 1234`
   *(Пароль сохранится в зашифрованную БД в `C:\ProgramData\MonitoringControl`)*

## Запуск для визуального тестирования
Для удобства в корне проекта лежат два скрипта:

### 1. Запуск Ядра (Core Service)
Ядро должно работать всегда.
1. Нажмите правой кнопкой мыши по файлу `run_core_admin.bat` и выберите **"Запуск от имени Администратора"**.
2. Откроется черное окно консоли. Вы увидите логи запуска базы данных и создания Named Pipes.
3. **Не закрывайте это окно!** Это и есть ваше работающее Ядро.

### 2. Запуск Трей-агента (Tray Agent)
Трей-агент — это UI для обычного пользователя (без прав админа).
1. Обычным двойным кликом запустите `run_tray.bat` (от имени администратора не нужно).
2. Обратите внимание на область возле часов (System Tray). Там появится иконка.
   * 🟢 **Зеленая иконка:** Трей нашел Ядро и оно активно.
   * 🔴 **Красная иконка:** Ядро не найдено (или вы закрыли консоль с Ядром).

### 3. Как протестировать логику
1. Нажмите правой кнопкой мыши на Зеленую иконку в трее.
2. Выберите **"Приостановить защиту"**.
3. Появится диалоговое окно ввода пароля. Введите `1234` (тот, что вы задали на этапе подготовки) и нажмите ОК.
4. Вы увидите, что иконка в трее поменяла цвет на 🟡 **Желтый** (статус SUSPENDED). 
5. В черном окне консоли Ядра появится лог: `System Suspend successfully activated via master password override.`
6. Снова нажмите ПКМ по иконке -> **"Возобновить защиту"** -> введите `1234`. Иконка снова станет зеленой.

Если вы принудительно закроете консоль с Ядром (нажмете крестик), Трей-агент в течение 2-х секунд поймет, что связь оборвалась, и перекрасит иконку в красный цвет.
